# Active vertex model with cell-cell viscous adhesion

This repository contains MATLAB code to simulate the shear-driven behavior of epithelial tissues using a Vertex-based model. The code supports research published in:

This model simulates cellular mechanics in the presence of cell-cell adhesion force, capturing the complex interplay between different aspect of cell-cell adhesion.

The primary script for the simulation is `run_sim.m`.

The code is tested with MATLAB R2021a and later versions.
